Bento Grid Design

06.06.2024

Diferente de layouts baseados em cards, a área e posição de cada célula define uma ilha de funcionalidade. Bento Grid Design. Bento grid é uma forma de organizar a UI em pequenas caixas ocupando quantidades diferentes de células, na horizontal, vertical ou ambas, de acordo com o conteúdo.
